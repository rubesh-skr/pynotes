```python
# Program 25: Palindrome String
def main():
    text = input("Enter string: ")
    
    rev = text[::-1]
    
    if text == rev:
        print("Palindrome")
    else:
        print("Not Palindrome")

main()
```

    Enter string:  madam
    

    Palindrome
